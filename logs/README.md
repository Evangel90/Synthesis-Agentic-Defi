# Feature Logs Index

This directory contains branch-specific collaboration logs. Each log documents the brainstorms, pivots, and milestones achieved during a specific task or feature development.

## **How to Log**
1. When starting a new branch (e.g., `feat/my-feature`), create a corresponding log file: `logs/feat-my-feature.md`.
2. Document every major human-agent interaction and technical decision.
3. Upon merging your branch into `main`, ensure your log content is also appended to the root `logs.md`.
