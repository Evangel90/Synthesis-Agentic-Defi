## **Collaboration Protocol**

1.  **Branching:** Every feature or task must be developed on a dedicated branch (e.g., `feat/uniswap-quote`).
2.  **Distributed Logs:** Each branch MUST maintain a unique log file in the `logs/` directory (e.g., `logs/feat-uniswap-quote.md`).
3.  **Amalgamation:** Upon merging a Pull Request, the branch's log content will be appended to the root `logs.md` to provide a complete, chronological record for the AI judges.
4.  **Credit:** All code contributions are logged to demonstrate meaningful agentic participation.
